# 網頁版貪食蛇遊戲

這是一個使用純 JavaScript 和 HTML Canvas 實現的貪食蛇遊戲。

## 遊戲特色

- 簡單易上手的貪食蛇遊戲
- 使用方向鍵控制蛇的移動
- 配有網格、分數顯示
- 支援暫停和繼續遊戲功能
- 遊戲結束時可以選擇重新開始或返回主頁面

## 如何遊玩

1. 使用方向鍵（↑, ↓, ←, →）控制蛇的移動
2. 按 "P" 鍵可以暫停遊戲
3. 遊戲暫停時，按空格鍵可以繼續遊戲
4. 遊戲結束時，可以按 "R" 重新開始，或 "Q" 返回主頁面

## 技術細節

- 採用純 JavaScript 實現
- 使用 HTML Canvas 繪製遊戲畫面
- 無需任何額外的框架或庫
